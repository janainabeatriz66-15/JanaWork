# JanaWork

**JanaWork** é uma plataforma local/offline de gestão para pequenos negócios e empresas de serviços.

O projeto nasceu a partir de uma ideia desenvolvida em um TCC, inspirada em uma necessidade real: organizar clientes, agenda, salas, contratos e financeiro em um único lugar, substituindo controles manuais e planilhas.
O projeto foi inicialmente desenvolvido como TCC e depois modernizado como uma plataforma funcional de gestão.

## ✨ Funcionalidades atuais

- Dashboard executivo
- Cadastro de clientes
- Agenda de reservas
- Gestão de salas
- Contratos
- Financeiro
- Exportação CSV
- Salvamento local no navegador
- Interface moderna em HTML, CSS e JavaScript

## 🖥️ Como executar

1. Baixe ou clone o repositório.
2. Abra o arquivo `index.html` no navegador.
3. Use o sistema localmente.

Não precisa instalar dependências, servidor, banco de dados ou hospedagem.

## 💾 Armazenamento

Os dados são salvos no próprio navegador usando `localStorage`.

Isso significa que o sistema funciona offline, mas os dados ficam vinculados ao navegador utilizado.

## 📌 Objetivo do projeto

O JanaWork foi criado como um projeto de portfólio para demonstrar:

- visão de produto;
- organização de processos;
- gestão de clientes;
- controle operacional;
- análise de dados;
- evolução de uma ideia acadêmica para uma solução funcional.

## 📖 Origem

A ideia surgiu a partir da experiência em um ambiente semelhante a coworking, onde reservas, clientes, contratos e controles financeiros eram feitos de forma manual ou em planilhas.

O projeto foi inicialmente desenvolvido como TCC e depois modernizado como uma plataforma funcional de gestão.
